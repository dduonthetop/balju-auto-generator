# 스미후루 발주서 생성기

접속 주소(GitHub Pages):  
https://dduonthetop.github.io/balju-auto-generator/

현재 웹페이지는 브라우저에서 직접 엑셀을 생성합니다.
즉, 위 주소만 열면 바로 사용할 수 있고 별도 API 서버 주소 입력은 필요 없습니다.

## 사용 방법

1. 기준 파일 1개 업로드
2. 참고 파일 여러 개 업로드
3. `발주서 생성` 클릭
4. 생성된 파일 다운로드

## Render 서버

`automatic_server.py`는 선택용 보조 서버입니다.
기본 사용에는 필요하지 않으며, GitHub Pages 주소만으로 충분합니다.
